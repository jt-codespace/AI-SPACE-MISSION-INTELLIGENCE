# 🚀 AI Space Mission Intelligence Dashboard

An AI-powered analytics platform for analyzing historical space missions, visualizing launch trends, and predicting mission outcomes using Machine Learning.

---

## 📌 Project Overview

This project combines Data Analytics, Machine Learning, and Dashboard Development to provide insights into historical space missions.

The system analyzes mission records from multiple space agencies and launch providers and generates visual insights regarding:

- Mission Success Rates
- Launch Trends
- Country-wise Launch Activity
- Top Space Organizations
- Mission Cost Analysis
- AI-Based Mission Success Prediction

---

## 🎯 Objectives

- Analyze historical space mission data
- Visualize mission trends and patterns
- Build a machine learning model for mission outcome prediction
- Create an interactive dashboard for data exploration
